# 🍳 Recipe Bot — Телеграм-бот для поиска рецептов

[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Телеграм-бот для поиска рецептов блюд по названию или ингредиентам через **TheMealDB API**.

Разработан в рамках финального проекта курса **Python-разработчик Skillbox**.

---

## 📋 Описание

Бот позволяет пользователям:
- 🔎 Искать рецепты по названию блюда (пицца, паста, курица...)
- 🥕 Искать рецепты по ингредиентам (сыр, томат, чеснок...)
- 📜 Просматривать историю своих запросов
- 🔗 Переходить на полные рецепты на сайте TheMealDB
- 🗑️ Очищать историю поиска

Поддерживает ввод на **русском** и **английском** языке.

## 🚀 Установка и запуск

### Шаг 1: Клонируйте репозиторий

```bash
git clone https://gitlab.skillbox.ru/stanislav_smirnov_2/python_basic_diploma.git
cd python_basic_diploma
```

### Шаг 2: Создайте виртуальное окружение

**Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

**Linux/Mac:**
```bash
python3 -m venv venv
source venv/bin/activate
```

### Шаг 3: Установите зависимости

```bash
pip install -r requirements.txt
```

### Шаг 4: Настройте переменные окружения

1.  Скопируйте шаблон:
    ```bash
    cp .env.template .env
    ```

2.  Откройте `.env` и вставьте ваш токен:
    ```
    BOT_TOKEN=1234567890:AABBccDDeeFFggHHiiJJkkLLmmNNooP
    ```

**Где получить токен:**
1.  Откройте [@BotFather](https://t.me/BotFather) в Telegram
2.  Напишите `/newbot`
3.  Следуйте инструкциям
4.  Скопируйте полученный токен

### Шаг 5: Запустите бота

```bash
python main.py
```

**Ожидаемый вывод:**
```
==================================================
🍳  RECIPE BOT — ЗАПУЩЕН!
==================================================
🤖 Бот готов к работе...
📊 База данных: подключена
🌐 Telegram API: подключено
==================================================
💡 Для остановки нажмите: Ctrl + C
==================================================
```

## 🌐 API

Проект использует **TheMealDB API** — бесплатную базу данных рецептов:

| Параметр | Значение |
|----------|----------|
| **Сайт** | https://www.themealdb.com/ |
| **Документация** | https://www.themealdb.com/api.php |
| **API Key** | Не требуется (бесплатный доступ) |
| **Лимит запросов** | Без ограничений |

### Эндпоинты:

| Метод | Описание | Пример |
|-------|----------|--------|
| `search.php` | Поиск по названию блюда | `?s=pizza` |
| `filter.php` | Поиск по ингредиенту | `?i=chicken` |
